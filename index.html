DevOps: Test Runner Setup

  • Mocha is a test framework. It is responsible to run our tests and report back which failed.
  • Chai is an assertion library. It provides syntactic sugar to check whether our code does the things we expect it to do.
  • Sinon
  • Babel lets us write ES6 code.
  • Istanbul generates our code coverage reports.
  • Codecov lets us upload our Istanbul code coverage reports.
  • Karma is a test runner. It allows us to run our client side JavaScript tests in real browsers from the command line.
  • Karma-mocha Reporter creates Karma test reports that look like Mocha's within the console.
  • Saucelabs
