DevOps: Test Runner Setup

  • Mocha is a test framework. It is responsible for running our tests and reporting back which failed.
  • Chai is an assertion library. It provides syntactic sugar that makes our Mocha tests look way sweeter.
  • Sinon
  • Babel transpiles our ES6 code into ES5 for compatibility purposes.
  • Istanbul generates our code coverage reports.
  • Codecov uploads our Istanbul code coverage reports to the cloud.
  • Karma is a test runner. It allows us to run our client side JavaScript tests in real browsers from the command line.
  • Karma-mocha Reporter creates Karma test reports that look like Mocha's within the console.
  • Saucelabs
